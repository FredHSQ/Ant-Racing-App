<!-- GETTING STARTED -->
## Inicializating project

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/FredHSQ/Ant-Racing-App/
   ```
3. Go to the project
   ```sh
   cd ant-racing
   ```
4. Install all packages
   ```sh
   npm install;
   ```
   or
   ```sh
   yarn install;
   ```
5. Start project
   ```sh
   npx expo start;
   ```
   then 
   read the QR code if you have the expo app installed
   
   or (if you have a android emulator)
   ```sh
   a;
   ```
